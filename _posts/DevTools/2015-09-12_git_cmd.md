# Git常用指令

`git config --global user.name "yourusername"` 設置使用者名  
`git config --global user.email "youremail@gmail.com"` 設置電子郵件  
`git init` 在指定目錄建立倉庫, 同時產生.git目錄  
`git add file.txt` 把檔案增加至暫存區  
`git commit -m "Comment"` 把文件提交至倉庫  
`git status` 查看提交狀態    
`git diff file.txt` 查看更變後的不同  
`git log` 查看歷史記錄  
`git log --pretty=oneline` 以簡短格式題示歷史記錄  
`git reset  --hard HEAD^` 返回到上個版本，^數量等同返回數量  
`git reset  --hard HEAD~100` 返回到指定版本   
`git reflog` 查看版本號  
`git reset  --hard 6fcfc89`　恢復至指定版本號(6fcfc89)  
`git checkout  --  file.txt` 撤消改變工作區的文件
